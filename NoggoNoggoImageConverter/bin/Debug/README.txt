Name: NoggoNoggoImageConverter
Autho: Tim Schneider

Description: Takes a picture and creates three matrices with rgb values.

How To Use: Select with the button labeled "..." the picture. Some statistics show up. To create the three textdokuments (for each color one), press "Convert". The files appear in the same folder as the source picture. The files are named as following: [picturename]_[color (RED, GREEN or BLUE)].txt.
